# tacokeeper.com

Source code of tacokeeper.com


## Development

Install the dependencies:

```sh
bundle install
```

Then serve the site:

```sh
# Available at http://127.0.0.1:4000/
bundle exec jekyll serve
```


## Scripts

```sh
cd twitter-bot
. export_credentials.sh

# Create a new user profile
./create_user_profile.py

# Process latest tweets
./process_tweets.py
```
